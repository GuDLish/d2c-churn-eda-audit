# d2c-churn-eda-audit
# D2C Customer Churn Capstone — Part 1: EDA & Business Understanding

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) and business understanding for a Direct-to-Consumer (D2C) customer churn prediction problem.

The goal of this phase is to:
- understand customer behavior,
- inspect data quality,
- identify churn patterns,
- analyze revenue trends,
- generate actionable business insights.

## Key Analysis Performed

### Data Quality Checks
- Missing value analysis
- Duplicate record analysis
- Datatype validation
- Dataset structure inspection

### Business Analysis
- Customer distribution analysis
- Revenue analysis
- Product category analysis
- Churn analysis
- Acquisition channel analysis

### Visualizations
- Customer distribution charts
- Revenue charts
- Churn distribution charts
- Churn rate comparisons

---

## Key Insights

- Skin Care is the highest revenue-generating category.
- Tier 1 cities contribute the largest customer base.
- Google Search and Instagram acquisition channels show higher churn rates.
- Overall churn rate is approximately 47%.
